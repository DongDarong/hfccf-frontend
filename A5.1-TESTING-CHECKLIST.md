# Phase A.5.1 — Testing Checklist

## Manual Testing Scenarios

### Scenario 1: Create New Monthly Submission
**Setup:** Teacher logged in, assigned to at least one class

**Steps:**
1. Navigate to `/module/preschool-admin/teacher/assessments`
2. Click "New Assessment" button
3. Select class from dropdown
4. Select assessment category
5. System shows current month
6. List of class students displays with empty score fields
7. Enter score for first student (e.g., 85.5)
8. Enter rating (e.g., "Good")
9. Click "Save Draft"
10. Verify toast: "Draft saved successfully"
11. Refresh page
12. Verify submission appears in list as "Draft"

**Expected Result:** ✅ Pass
- [ ] New submission created in backend
- [ ] Scores persisted
- [ ] Submission appears in list with 0% progress initially
- [ ] Can edit again without errors

---

### Scenario 2: Submit Completed Assessment
**Setup:** Submission created with at least one student score

**Steps:**
1. Open submission detail
2. Fill in scores for all students
3. Click "Save Draft"
4. Verify progress shows 100%
5. Click "Submit for Review"
6. Confirm dialog appears
7. Click "OK" to confirm
8. Verify redirect to list
9. Verify status changed to "Submitted"
10. Attempt to click edit button
11. Verify edit still works (allows resubmit from returned)

**Expected Result:** ✅ Pass
- [ ] API receives POST to /submit endpoint
- [ ] Status in list updates immediately
- [ ] Button no longer shows "Submit for Review"
- [ ] Notification confirms submission

---

### Scenario 3: Resubmit After Return
**Setup:** Submission with status "Returned" (manually set via admin or API)

6. Re-enable network
7. Retry operation
8. Verify works correctly on retry

**Expected Result:** ✅ Pass
- [ ] Error caught and displayed gracefully
- [ ] No console errors visible to user
- [ ] UI remains responsive
- [ ] Can retry without page reload

## Automated Testing Scenarios

### API Integration Tests
```javascript
describe('Monthly Assessment Workflow', () => {
  it('loads submissions list', async () => {
    const result = await fetchMonthlySubmissions()
    expect(result.items).toBeDefined()
    expect(result.pagination).toBeDefined()
  })

  it('creates new submission', async () => {
    const submission = await createMonthlySubmission({
      academic_year_id: '2026-ay',
      class_id: 'class-1',
      assessment_category_id: 'cat-1'
    })
    expect(submission.id).toBeDefined()
    expect(submission.status).toBe('draft')
  })

  it('updates student score', async () => {
    const assessment = await updateMonthlySubmissionScore(
      'submission-1',
      'student-1',
      { score: 85.5, rating: 'good' }
    )
    expect(assessment.score).toBe(85.5)
  })

  it('submits for review', async () => {
    const submission = await submitMonthlySubmission('submission-1')
    expect(submission.status).toBe('submitted')
  })
})
```
